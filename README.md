# HTML Authentication POC

## Overview

This project is a simple HTML-based Authentication System created using only basic HTML structure and anchor tag redirections. No CSS or JavaScript is used.

## Files Included

- login.html  
- register.html  
- forgot-password.html  
- reset-password.html  
- dashboard.html  

## Page Flow

- Login → Dashboard  
- Register → Login  
- Forgot Password → Reset Password → Login  
- Dashboard → Logout → Login  

All navigation is handled using anchor tags.

## How to Run

1. Open `login.html` in a web browser.
2. Navigate through the pages using the provided links.
3. Verify all redirections work correctly.

## Technologies Used

- HTML5  
- Anchor tags for navigation  

This repository is public for evaluation purposes.
